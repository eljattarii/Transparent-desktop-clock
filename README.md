# 🕒 Transparent Desktop Desklet Clock

A sleek, modern, and highly optimized transparent desktop clock designed for Linux systems (perfectly tailored for Linux Mint). Built using **Python 3** and **PyQt6**, this widget blends beautifully into your desktop environment while keeping resource consumption at absolute zero.

---

## ✨ Features

- **True Transparency & Frameless:** Zero borders, completely translucent background that integrates natively with your desktop wallpaper.
- **0% CPU Smart Timer:** Unlike standard desktop clocks that refresh every second, this clock calculates the exact milliseconds remaining until the next minute and goes to sleep. It only wakes up for a fraction of a millisecond to update the time, ensuring your CPU stays cool and fans remain silent.
- **Smart Drag & Docking:** You can drag the clock anywhere on your screen. It automatically detects your screen boundaries and prevents itself from clipping outside the visible area.
- **Persistent Position Memory:** Instantly saves its last coordinates to a local `config.ini` file upon release, so it spawns exactly where you left it on the next boot.
- **Robust Path Resolution:** Programmatically locks its working directory, making it immune to path-resolution breaks when triggered by system startup daemons.

## 🛠️ Requirements

- **OS:** Linux (Tested on Linux Mint / Ubuntu)
- **Python:** Python 3.8 or higher
- **Dependencies:** PyQt6

## 🚀 Installation & Setup

Follow these steps to get the clock running in an isolated virtual environment (`venv`) to keep your system clean.

### 1. Clone the Repository
```bash
git clone https://github.com/eljattarii/transparent-desktop-clock.git
cd transparent-desktop-clock


```
*(Make sure to put your actual Poppins-Black.ttf font file inside this directory)*
### 2. Create and Activate a Virtual Environment
```bash
python3 -m venv venv
source venv/bin/activate

```
### 3. Install Requirements
```bash
pip install --upgrade pip
pip install PyQt6

```
### 4. Run the Application
```bash
python3 transparent_clouck.py

```
## ⚙️ How to Make it Run Automatically at Startup (Linux Mint)
To make the clock start beautifully 13 seconds after your desktop environment fully loads (preventing any CPU spikes during initial boot):
 1. Open your terminal inside the project directory (ensure your venv is active) and run this command to generate your exact absolute execution path:
   ```bash
   echo "$(which python3) '$(pwd)/transparent_clouck.py'"
   
   ```
 2. Copy the full output line generated by the terminal.
 3. Open the **Startup Applications** utility from your Linux Mint application menu.
 4. Click the **+** (Add) button at the bottom and select **Custom Command**.
 5. Fill in the fields as follows:
   * **Name:** Desktop Clock
   * **Command:** *Paste the exact line you copied in Step 2 here (Ensure paths with spaces are enclosed in double quotes "")*
   * **Startup Delay:** Set it to 10 seconds.
 6. Click **Save** and reboot your system!
