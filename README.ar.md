# 🕒 ساعة سطح المكتب الشفافة.
ساعة سطح المكتب شفافة، عصرية، ومصممة بأعلى درجات الكفاءة لأنظمة Linux (ومهيأة خصيصاً لتوزيعة Linux Mint). تم بناؤها باستخدام **Python 3** ومكتبة **PyQt6**، لتندمج بشكل جمالي رائع مع بيئة سطح المكتب لديك مع الحفاظ على استهلاك موارد الجهاز عند الصفر مطلقاً.

---

## ✨ المميزات

- **شفافية كاملة وبدون إطار (True Transparency & Frameless):** بدون أي حواف، مع خلفية شفافة تماماً تندمج بشكل طبيعي مع خلفية سطح المكتب الأصلية.
- **مؤقت ذكي باستهلاك 0% للمعالج (0% CPU Smart Timer):** على عكس ساعات سطح المكتب التقليدية التي تقوم بتحديث الوقت كل ثانية، تحسب هذه الساعة أجزاء الثانية المتبقية بدقة حتى بداية الدقيقة التالية ثم تدخل في وضع الخمول التام (Sleep). تستيقظ الساعة فقط لجزء من الملي ثانية لتحديث الأرقام، مما يضمن بقاء المعالج بارداً والمراوح صامتة تماماً.
- **سحب وتثبيت ذكي (Smart Drag & Docking):** يمكنك سحب الساعة لأي مكان على شاشتك. يقوم البرنامج تلقائياً باكتشاف أبعاد الشاشة ومنع الساعة من الخروج عن الحواف المرئية.
- **ذاكرة حفظ الموقع (Persistent Position Memory):** تحفظ الساعة إحداثيات موقعها الأخير فوراً في ملف `config.ini` محلي عند إفلاتها بالماوس، لتظهر في نفس المكان تماماً عند إعادة التشغيل القادمة.
- **معالجة قوية لمسارات النظام (Robust Path Resolution):** يقوم البرنامج برمجياً بقفل مسار العمل الأساسي له، مما يجعله محصناً ضد أخطاء ضياع مسارات الملفات (مثل ملف الخط) عند تشغيله تلقائياً بواسطة برامج بدء التشغيل بالنظام.

## 🛠️ المتطلبات

- **نظام التشغيل:** Linux (تم اختبارها على Linux Mint / Ubuntu)
- **لغة بايثون:** Python 3.8 أو أعلى
- **المكتبات الخارجية:** PyQt6

## 🚀 التثبيت والإعداد

اتبع هذه الخطوات لتشغيل الساعة داخل بيئة افتراضية معزولة (`venv`) للحفاظ على نظافة واستقرار نظامك.

### 1. استنساخ المستودع (Clone)
```bash
git clone https://github.com/eljattarii/transparent-desktop-clock.git
cd transparent-desktop-clock


```
*(تأكد من وضع ملف الخط الخاص بك باسم Poppins-Black.ttf داخل هذا المجلد)*
### 2. إنشاء وتفعيل البيئة الافتراضية
```bash
python3 -m venv venv
source venv/bin/activate

```
### 3. تثبيت المكتبات المطلوبة
```bash
pip install --upgrade pip
pip install PyQt6

```
### 4. تشغيل التطبيق
```bash
python3 transparent_clock.py

```
## ⚙️ طريقة التشغيل التلقائي عند إقلاع الحاسوب (Linux Mint)
لجعل الساعة تبدأ بالعمل تلقائياً وبسلاسة بعد 13 ثانية من تحميل واجهة سطح المكتب بالكامل (لمنع أي ضغط على المعالج أثناء الإقلاع الأولي للنظام):
 1. افتح الطرفية (Terminal) داخل مجلد المشروع (تأكد من تفعيل البيئة الافتراضية venv) ونفذ هذا الأمر لتوليد مسار التشغيل المطلق الخاص بجهازك بدقة:
   ```bash
   echo "$(which python3) '$(pwd)/transparent_clock.py'"
   
   ```
 2. قم بنسخ السطر الكامل الناتج في الطرفية.
 3. افتح أداة **تطبيقات بدء التشغيل (Startup Applications)** من قائمة التطبيقات في Linux Mint.
 4. اضغط على زر الزائد **+** في الأسفل واختر **أمر مخصص (Custom Command)**.
 5. املأ الخانات كالتالي:
   * **الاسم (Name):** Desktop Clock
   * **الأمر (Command):** *قم بلصق السطر الذي نسخته تماماً من الخطوة رقم 1 هنا (تأكد من إحاطة المسارات التي تحتوي على مسافات بعلامات اقتباس مزدوجة "")*
   * **تأخير بدء التشغيل (Startup Delay):** اضبطه على 10 ثانية.
 6. اضغط على **حفظ (Save)** وأعد تشغيل جهازك للاستمتاع بالنتيجة!
